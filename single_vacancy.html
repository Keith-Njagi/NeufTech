SENIOR FRONTEND DEVELOPER

Recruitment ends on: 21st March, 2021
Description

We are looking for a senior JS Frontend developer to join our Engineering team. We’re building software products with a variety of technologies on the backend (Javascript, Python), and mostly doing React (with ES6) front-ends.

If this resonates, then please keep reading to learn if you are a match for this permanent, full time, remote position.

 

Responsibilities

 

● Implement new features and enhance the user experience across a suite of applications

● Ensure code quality through unit testing and peer code review

● Push code and see it go live the same week

● Work with a Product Manager and UX/UI Designer to refine scope

● Occasionally assist with the investigation and resolution of support issues ● Routinely give time estimates for what you're working on

● Ability to lead a small team of developers

The way we work:

We work flexibly. As such, we are well equipped to work effectively in the current global landscape through strong communication and organization, even with team members across several time zones. Our core team has key people in other parts of the world including USA, Kenya, India and canada.

Our day-to-day work leverages a Scrum-style rhythm using the latest project management tools, CI/CD, and Slack for team communication.

Qualifications

REQUIREMENTS:

Core requirements:

● Bachelor’s degree or higher in Computer Science or equivalent.

● At least 5 years of professional experience

● Good communication skills in english

● 4 years experience using React or comparable front-end frameworks

● Proficient with Javascript

● Experience with CSS Preprocessors (SASS or LESS) or CSS-in-JS libraries like Styled Components

● experience using git or similar VCS

● experience writing unit tests and End to End test

● experience using secure, RESTful web APIs

● experience using NPM and various Node.js APIs

● Strong aesthetic sensibilities

● Full-time availability is a strong advantage

Extra credit:

● Experience working with python

● Do you contribute to open source projects or have a passion project? Tell us about it!
